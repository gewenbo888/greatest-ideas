# The Greatest Ideas · 最伟大的想法

An opinionated, bilingual canon of the **64 ideas** that most changed what
humans can do or know — from fire to transformers, across 8 epochs, with a
live D3 lineage graph showing which idea spawned which.

> Greatness is not loudness or proof. It is fertility — how many other
> thoughts a thought permits to exist.

Next.js 14 (App Router) · TypeScript · Tailwind · Framer Motion · D3 · Cormorant Garamond / Spectral / Noto Serif SC.
Codex / parchment aesthetic with a vermilion accent; canvas ember in the hero;
SVG glyphs hand-curated per idea.
